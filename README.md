# Setup Instructions

## Prerequisites
- Node.js (version >= 14)
- npm (version >= 6)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Shen212/Design-Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Design-Portfolio
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To start the project, run:
```bash
npm start
```
